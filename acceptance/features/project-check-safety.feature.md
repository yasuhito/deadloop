# 機能: Git 管理ファイルが deadloop の作業用一時ディレクトリにある場合、自動チェックを実行しない

deadloop は、自動チェックの前に、作業用一時ディレクトリにある
Git 未追跡ファイルを一時的にリポジトリの外へ移す。

ただし、作業用一時ディレクトリに Git 管理ファイルがある場合は、
必要なファイルまで移す可能性があるため、自動チェックを実行しない。

## シナリオ: 作業用一時ディレクトリに Git 管理ファイルがある

* 前提 deadloop の作業用一時ディレクトリに Git 管理ファイルがある
* もし deadloop が自動チェックを開始しようとする
* ならば deadloop は自動チェックを実行しない
